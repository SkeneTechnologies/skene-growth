# Generate Growth Plan

1. Invoke the `plan-growth-loop` skill.
2. Run `uvx skene plan --help` first to discover available flags.
3. **Run autonomously.** Present results, help user pick a loop (the one decision point), store selection. Suggest `/skene-build`.
